# Guanacaste Development Group

Marketing site for Guanacaste Development Group — real estate development and
investment in Guanacaste, Costa Rica.

Built with Next.js (App Router, static export). Produced end-to-end by the
EKO-I autonomous build pipeline; quality-gated for accessibility (WCAG AA),
W3C validity, Core Web Vitals, and bilingual copy.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npx next build   # static export to ./out
```

Deployed on Vercel.
